SolarIA : Bachelor Degree Project

IOT Solar Tracker : SolarIA

SolarIA is an advanced IoT ecosystem engineered to simplify the adoption of solar energy technologies. Leveraging AI-powered solar array budgeting and intelligent tracking devices, it optimizes energy absorption using geospatial data, ensuring maximum efficiency and cost-effectiveness for sustainable power solutions.

Components

Hardware

Featuring a steel/aluminum frame capable of hold a 160W solar panel, and ball-bearing axis for tilt and rotation, movement is assisted by servo motors and 3D-printed gears. Additionally, a daylight and infrared presence sensor allows the detection of shadows or obstructions on the solar panel. Servo motors and sensors are controlled by a Raspberry Pi. A scalable architecture model integrating sensors and servo motors, capable of managing up to 32 devices per processing unit.

Software

Client-server infrastructure design for a web-based application featuring API integration with OpenAI for area estimation using satellite imagery (Google Maps) and weather/sun data based on geolocation (OpenWeatherMap, AstroPy). Budget estimation based on location, area, and annual energy consumption analysis. Active monitoring of intelligent solar tracker arrays with real-time status notifications and alerts. A historical energy absorption dashboard provides detailed reporting and insights.

IoT Deployment

Implementing a robust client-server infrastructure with Git version control. The front-end is built using React and TypeScript, while the back-end leverages Python for seamless functionality. Cloud services are deployed via Microsoft Azure, with CI/CD pipelines ensuring automated workflows. Raspberry Pi 4 is used for device management, integrating a MongoDB database to store historical energy data and user installation records.
